# 📌 Project Overview

This project demonstrates the implementation and analysis of the **Discrete Fourier Transform (DFT)** and **Fast Fourier Transform (FFT)** using MATLAB. A discrete-time signal of length **N = 16** is generated, and its frequency spectrum is computed using both a manually constructed DFT matrix and MATLAB’s built-in FFT function.

The project validates the equivalence of DFT and FFT while highlighting the computational efficiency of FFT over direct DFT computation.

---

# 📖 Description

The analysis explores important frequency-domain concepts such as:

* Zero-padding
* Spectral leakage
* Windowing techniques

Zero-padding is applied to improve spectral resolution for visualization. Signals with integer and non-integer frequencies are used to demonstrate the effects of spectral leakage.

A **Hamming Window** is then applied to reduce leakage and smooth discontinuities at signal boundaries.

The simulation generates multiple plots to:

* Compare FFT and manual DFT results
* Visualize the impact of zero-padding
* Study spectral leakage before and after windowing

Performance comparisons between DFT and FFT are also included, demonstrating the computational complexity difference:

* **DFT → O(N²)**
* **FFT → O(N log N)**
